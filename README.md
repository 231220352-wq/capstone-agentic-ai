# UniGuide AI – Agentic University Assistant

## Overview
UniGuide AI is an intelligent agentic assistant designed to help university students access academic information quickly and efficiently. The system uses Retrieval-Augmented Generation (RAG) with multiple knowledge sources to answer student questions accurately while demonstrating modern Agentic AI concepts.

This project was developed as the capstone project for the **Building Agentic AI Systems** course.

---

## Features

- Multi-source knowledge retrieval (RAG)
- Intelligent routing between different knowledge sources
- Semantic search using embeddings
- Context-aware conversations
- LangGraph workflow
- Human-in-the-loop support
- Error handling and workflow management

---

## How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Add your OpenAI API key.
4. Run all notebook cells in order.
5. Ask questions through the assistant interface.

---

## Agent Architecture

The project follows a **Multi-Source Knowledge Base with Routing** architecture.

The assistant:
- Receives the user's question.
- Routes the request to the appropriate knowledge source.
- Retrieves relevant documents using semantic search.
- Generates an accurate response using Retrieval-Augmented Generation (RAG).
